⚡ Quick Start

Ikuti langkah berikut untuk mulai menggunakan API RFSANZ ERP dalam hitungan menit.

ℹ️
Info Base URL: http://localhost:6000 (development). Semua endpoint diawali /api/, kecuali Fleetbase adapter (/int/v1/).

Endpoint Utama

MethodEndpointKeterangan
POST/api/auth/loginKirim email & password → dapat JWT token
GET/api/auth/meProfil user yang sedang login
POST/api/auth/refreshPerbarui access token
POST/api/auth/logoutLogout dan invalidasi token
GET/api/healthStatus server & database

🔐 Autentikasi

RFSANZ ERP menggunakan JWT Bearer Token. Semua request ke endpoint protected harus menyertakan header Authorization.

⚠️
Perhatian Semua endpoint (kecuali /api/auth/login) memerlukan header: Authorization: Bearer <your_jwt_token>

Contoh Login Request

http 
POST /api/auth/login
Content-Type: application/json

{
  "email": "admin@rfsanz.com",
  "password": "your_password"
}

// Response 200 OK
{
  "accessToken":  "eyJhbGciOiJIUzI1NiJ9...",
  "refreshToken": "eyJhbGciOiJIUzI1NiJ9...",
  "user": { "id": "uuid", "name": "Admin", "role": "ADMIN" }
}
💡
Tip Gunakan tombol Authorize di tab API Explorer untuk menyimpan token dan mencoba semua endpoint langsung dari browser.

JWT_REFRESH_SECRET—Secret key untuk refresh tokenWajib
ADMIN_PASSWORD—Password akun admin defaultWajib
